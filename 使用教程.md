# 🌌 Clay SpaceStation | 星壤空间站 详细使用教程

本指南将协助您基于 **Clay-SpaceStation** 仓库构建属于自己的沉浸式个人主页。由于本项目采用 **Build-less (无构建)** 架构，您无需安装 Node.js、npm 或执行任何复杂的打包命令，只需修改 HTML 文件即可完成个性化配置。

---

## 🛠️ 第一步：开发环境准备

虽然本项目可以直接双击运行，但为了获得最佳开发体验，建议：
1. **编辑器**：推荐使用 [Visual Studio Code](https://code.visualstudio.com/)。
2. **预览工具**：在 VS Code 中安装 **Live Server** 扩展。这能确保根目录的重定向逻辑（多语言自动跳转）在本地能够准确执行。
3. **资源托管**：准备一个图床（如 GitHub 仓库或阿里云 OSS），用于托管您的头像和角色图片。

---

## 🎨 第二步：内容高度自定义

项目的所有核心数据都硬编码在各语言目录（如 `zh-CN/index.html`）的 `<script type="text/babel">` 区域内。

### 1. 修改护照与基础信息
在代码中搜索 `Profile` 组件或相关常量：
* **头像与姓名**：修改 `Hero` 组件中的图片链接和 `Claylark` 文本。
* **护照号码**：在 `Profile` 组件内修改 `Passport No.`（如 `CS91726710`）。
* **签发日期**：护照中的 `Date of Issue` 会自动获取您的本地系统当前日期。

### 2. 更新属性、游戏与项目经历
找到对应的数组常量进行替换：
* **MAIN_CHARACTERS / SUB_CHARACTERS**：更新角色名称及其对应的 `img`（图片）或 `video`（webm格式视频）链接。
* **GAMES / PROJECTS**：修改游戏或项目名称、起止时间及背景图链接。
* **SOCIAL_LINKS**：更新您的社交账号 ID、配色方案及跳转 URL。
* **EMAILS**：在 `EMAILS` 数组中填入您的电子邮箱地址。

### 3. 启用 AI 助手 (ClaySeek)
若要让右下角的 AI 助手生效：
1. 打开 `index.html`。
2. 搜索 `const ChatBot = () => {`。
3. 找到 `const apiKey = "";`，在引号内填入您的 **Google Gemini API Key**。
4. **语气调整**：您可以修改 `systemPrompt` 变量，为您的 AI 助手设定特定的性格和回复偏好。

---

## 🌍 第三步：多语言分发逻辑

根目录下的 `index.html` 负责根据访客的浏览器设置自动路由：
* **自动跳转**：脚本会读取 `navigator.language`。
* **路由规则**：
    * `en` 开头：跳转至 `/en-US`。
    * 包含 `tw`：跳转至 `/zh-TW`。
    * 包含 `hk` 或 `mo`：跳转至 `/zh-HK`。
    * 默认/其他：重定向至 `/zh-CN`。

如果您需要新增语言版本，请在此处修改 `targetPath` 判断逻辑。

---

## 🚀 第四步：部署上线

本项目是纯静态文件，部署极其简单：

### 方法 A：GitHub Pages
1. 创建一个新的 GitHub 仓库并推送代码。
2. 进入仓库的 **Settings > Pages**。
3. Source 选择 **Deploy from a branch**，Branch 选择 `main` 分支。
4. 保存后稍等片刻，即可通过 `https://<your-username>.github.io/<repo-name>` 访问。

### 方法 B：Vercel / Cloudflare Pages
1. 直接导入您的 GitHub 仓库。
2. 部署设置中，框架预设选择 **Other** (或无需设置)。
3. 点击 **Deploy** 即可完成自动化部署。
